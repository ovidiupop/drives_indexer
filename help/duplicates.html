Duplicates

In the Duplicates tab, you can find duplicates from indexed files.

Press the "Find" button. The duplicates will be identified based on the file size and the filename.

The table will be populated with found duplicated. In the last column, you can check the boxes for the duplicates you wish to remove. By default, there will be a reference file which will not be checked and the next duplicates will be checked.
After you selected the instances of the files which you wish to remove, you can export the list as csv.
