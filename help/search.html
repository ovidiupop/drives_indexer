Search

In the Search tab you can find the files you want!

If you have indexed your folders, enter your search term, press Enter, or click the Search button.

If you wish to search only for a specific category of files, you can uncheck the rest of categories.

You can sort results as you wish, clicking on the table's header.

If some of the results belong to a drive which is not mounted, the name of that drive will be red.

If the results belong to a mounted drive, with a double click on the table line you can quickly see information about each file. Many files also have preview.
